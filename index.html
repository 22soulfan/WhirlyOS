How to Install

‚ö†Ô∏è Warning & Liability

Install at your own risk. The owner of WhirlyOS is not responsible for any hardware damage or warranty void. Be careful when partitioning drives for dual boot.

On Windows:

  1. Download bootable USB creator (Rufus/Etcher).
  2. Download WhirlyOS ISO from repository releases.
  3. Flash ISO to USB drive.
  4. Restart computer and enter Boot Manager (Del, F10, F12).
  5. Disable Secure Boot if necessary.

On Linux (dd command):

sudo dd if=path/to/whirlyOS.iso of=/dev/sdx bs=4M status=progress conv=fsync

Note: It is recommended to use BalenaEtcher if you are not advanced with terminal commands.
