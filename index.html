How to Install

Disclaimer: Install at your own risk. The owner is not responsible for damage or warranty void. Back up your original OS first.

On Windows

  1. Download Rufus or BalenaEtcher.
  2. Download WhirlyOS ISO from releases.
  3. Select USB and ISO in creator, click Start/Flash.
  4. Restart and enter Boot Manager (F10, F12, or Del).
  5. Disable Secure Boot if necessary to avoid violations.

On Linux (Terminal)

Note: Use dd only if you are an advanced user. Etcher is recommended.

sudo dd if=path/to/whirlyOS.iso of=/dev/sdx bs=4M status=progress conv=fsync
