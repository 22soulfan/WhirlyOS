Project Philosophy

Myriad: An indefinitely great number; having countless variations.

The Myriad Project is a multi-kernel universe curated by **22soulfan**. It wraps powerful Linux and NT bases in cinematic, ADHD-aware shells designed to help users discover their "Spark" through technology while maintaining digital health via built-in screen time protections.

The OS Matrix

Closed Source Flagship

MyriadOS

The proprietary surgical modification of the NT kernel. Achieves sub-200MB idle RAM while maintaining full binary compatibility with Windows applications. Features the most advanced "Spark Shell" in existence.

Open Source (WonderRepo)

WhirlyOS

The Debian-based educational powerhouse. Designed for i386 compatibility and schools. Features cinematic desktop environments and integrated GCompris/Scratch learning suites.

Windows 11 Alternative

openMyriad

An open initiative to replicate the Windows 11 UX on an open-source kernel. Eliminates TPM requirements and mandatory cloud accounts.

The System Bridge

Communiverse11

The ultimate community modification layer. Communiverse11 allows existing Windows 11 users to "bridge" their system into the Myriad universe, applying project-specific optimizations, privacy hardening, and cinematic themes without a full reformat.

Safety First

Install at your own risk. The Myriad Project is not responsible for data loss. Always perform a clean backup before partitioning.

Installation: Windows & Rufus

1. Download the ISO from WonderRepo.

2. Open Rufus and select your USB drive.

3. Select the ISO and click Start.

4. Disable Secure Boot in BIOS and boot from USB.

Installation: Linux & dd Command
sudo dd if=whirlyos.iso of=/dev/sdx bs=4M status=progress conv=fsync
