The OS Collection

WhirlyOS (Unix)

The education flagship. Based on Debian Stable, stripped of 80% of standard services for raw speed on legacy processors.

Knowledge Base

Kernel Comparison

WhirlyOS uses a Long Term Support (LTS) Linux kernel, while MyriadOS utilizes a custom NT-Clone kernel for driver parity.

The Spark Shell Architecture

Spark-Shell is written in C++ and Lua, allowing it to run as a native window manager on Linux and a shell replacement on Windows.

Installation Protocol

1

Create Media
