One Vision. Many Foundations.

Founded and maintained by **Hiro Hamada**, The Myriad Project is a solo endeavor to redefine personal computing. By forking and modernizing existing foundations like **ReactOS** and **Debian**, Hiro aims to provide specialized environments for different user needs.

Whether you are seeking the cinematic, ADHD-aware focus of **WhirlyOS** or the open-source Windows compatibility of **openMyriad**, this project puts user sovereignty first.

The Portfolio

Modern Windows

openMyriad

ReactOS Fork / Modern Win 11 Inspired

The open-sourced rebirth of Windows 11. Features an Android-inspired lockscreen and a modernized UI shell that stays true to the Windows 11 aesthetic while remaining completely open-source.

Educational Linux

WhirlyOS

Debian Stable / ADHD-Aware

A playful, educational distro featuring the SoulFrame desktop. Designed to help users "Find their Spark" with built-in distraction filters and cinematic UI.

ChromiumOS Hybrid

ChromMyriad

Chromium Base / Android Lock

Cloud speed with a twist. Integrates a native Android lockscreen into the ChromiumOS environment for a seamless mobile-to-desktop transition.

Minimal Win11

Communiverse11

Windows 11 Mod / Minimal Support

The "Ambassador" project. A stripped-down Windows 11 for users who want a minimal footprint, zero telemetry, and complete local control.
