One Vision. 22soulfan.

The **Myriad Project** is a solo initiative by **22soulfan**. This project is a curated collection of operating systems designed to bridge the gap between Windows 11 familiarity and open-source sovereignty.

Whether it is the ADHD-aware design of **WhirlyOS** or the open-sourced NT rebirth of **openMyriad**, every line of code and UI element is crafted to help users "Find their spark."

System Portfolio

Educational Linux

WhirlyOS

Debian Stable // SoulFrame

Playful, ADHD-aware shell for learning. Features screen-time limits and six cinematic editions. Focused on i386 compatibility.

Open Source NT

openMyriad

ReactOS Fork // C++ Shell

Modernized Windows rebirth with Android-inspired lockscreens and a hybrid Windows 10/11 start menu logic.

Cloud Hybrid

ChromMyriad

ChromiumOS // Aura Shell

Specialized ChromiumOS project featuring a native Android-style lockscreen for cloud sovereignty and minimal footprint.

Minimal Windows

Communiverse11

Win11 Mod // Ambassador Login

Stripped Windows 11 for minimal support, zero telemetry, and the security-focused Ambassador login role.
