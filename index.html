The Core

MyriadOS

The technical blueprint for the entire multiverse. MyriadOS standardizes the SoulFrame desktop experience, creating a modular UI logic that syncs configuration across different kernels.

Performance

renardOS

Inspired by the swift fox. renardOS is a hyper-minimalist environment that strips away background bloat to provide a lightning-fast workspace for developers.

Expressive

RPandaOS

A design-forward branch for vibrant UI exploration. RPandaOS focuses on playful interaction and touch-friendly widgets for creative professionals.

NT Rebirth

openMyriad

The modern open-source Windows 11 alternative. Built on ReactOS, featuring an Android-inspired lockscreen and modern C++ shell logic.

Educational

WhirlyOS

ADHD-aware, cinematic Linux (Debian) for students. Includes legacy i386 support and specialized Pixar-inspired desktop editions.

Cloud

ChromMyriad

ChromiumOS optimized with a native Android lockscreen. A bridge between cloud speed and mobile security aesthetics.
