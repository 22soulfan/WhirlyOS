What is Myriad?

"Myriad" (adj.) - An indefinitely great number; innumerable; having countless variations.

In the context of this project, **Myriad** represents the limitless potential of computing. It signifies the countless ways we can reinvent hardware, the innumerable paths a user can take to find their own digital "Spark," and the vast collection of operating kernels we maintain to ensure no machine is left behind. We call it The Myriad Project because there is no single solution to computing; there is only a myriad of possibilities.

System Hierarchy

Flagship / Closed Source

MyriadOS

The core flagship of the project. MyriadOS is a proprietary, highly optimized version of the NT kernel. It features the most advanced version of the "Spark" desktop environment, written in pure C++ for maximum performance. It is designed to be the ultimate daily driver for power users, offering full binary compatibility with Windows applications while maintaining absolute control over system resources.

Community / Open Source

WhirlyOS

The educational Unix backbone. Based on Debian, WhirlyOS is the open-source gateway for users learning about system optimization. It is the fastest-booting system in our collection, focusing on sub-1GB RAM stability and educational tools.

Initiative / Open Source

openMyriad

Our answer to modern accessibility. openMyriad is an open-source project focused on rebuilding the Windows 11 experience from the ground up using open kernels. It provides a familiar UI but removes the telemetry and hardware restrictions (No TPM required).

Cloud / Hybrid

ChromMyriadOS

The web-first operating system. Built on a Gentoo Linux base, ChromMyriadOS is designed for Chromebooks and web-heavy users. It integrates the Myriad Project aesthetic into a browser-centric workflow for maximum efficiency.

Mod / Community

Communiverse11

The community modification of the standard NT kernel. Communiverse11 acts as a "bridge" system, allowing users to modify existing Windows 11 installations to adopt Myriad Project features and privacy settings without a full reformat.
