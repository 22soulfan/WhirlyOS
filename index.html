Install at your own risk. Improper installation may void warranty. The creator of WhirlyOS is not responsible for hardware damage. Make sure to have a backup of your original OS.

Windows Installation Steps:

  1. Download bootable USB creator (Rufus/Etcher).
  2. Download WhirlyOS ISO from repository releases.
  3. Flash ISO to USB drive.
  4. Restart and enter Boot Manager (F10, F12, or Del).
  5. Disable Secure Boot if necessary to avoid security violations.

Linux (dd command):

sudo dd if=path/to/whirlyOS.iso of=/dev/sdx bs=4M status=progress conv=fsync
