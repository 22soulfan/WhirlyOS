Installation Guide

WARNING: Install at your own risk. Warranty may void. The owner is not responsible for any damage. Shrink partitions carefully for dual-boot.

Windows

Rufus, BalenaEtcher, or Win32Imager. Disable Secure Boot if necessary.

Linux

Recommended: Etcher. Advanced users: Use dd command.

DD Command Usage:

sudo dd if=path/to/whirlyOS.iso of=/dev/sdx bs=4M status=progress conv=fsync
