Installation Guide

‚ö†Ô∏è Warning

Install at your own risk. The owner of this OS is not responsible for damage. Make sure to back up your original operating system. Dual boot requires shrinking drives‚Äîbe careful!

Windows & MacOS

  1. Download **Rufus** (Windows) or **BalenaEtcher** (Windows/Mac).
  2. Download WhirlyOS ISO from GitHub releases.
  3. Flash ISO to a USB drive.
  4. Restart and enter Boot Manager (Del, F10, or F12).
  5. Disable **Secure Boot** if you encounter violations.

Linux Terminal (Advanced)

Recommended to use Etcher, but you can use `dd`:

sudo dd if=whirlyOS.iso of=/dev/sdx bs=4M status=progress conv=fsync

Ensure /dev/sdx is your correct USB path.
