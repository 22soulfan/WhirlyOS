Installation Protocol

‚ö†Ô∏è Critical Warning

Install at your own risk. Warranty may void. The creator of WhirlyOS is not responsible for hardware damage. Always backup your data before partitioning.

Windows / MacOS

  1. Download BalenaEtcher or Rufus.
  2. Download the WhirlyOS ISO.
  3. Flash the image to a USB drive.
  4. Restart and enter Boot Manager (F10/F12).
  5. Disable Secure Boot in BIOS.

Linux Terminal

Recommended for advanced users:

sudo dd if=whirlyOS.iso of=/dev/sdx bs=4M status=progress conv=fsync
