Install at your own risk. Improper installation may result in warranty void or data loss. The creator and The Myriad Project are not responsible for damage.

Windows Installation

  1. Download ISO from repository releases.
  2. Use Rufus or BalenaEtcher to flash the image to USB.
  3. Restart and enter Boot Manager (Del, F10, or F12).
  4. Disable Secure Boot to avoid security violations.

Linux (Advanced)

Recommended: Use Etcher for safety. For CLI, use dd:

sudo dd if=whirlyOS.iso of=/dev/sdx bs=4M status=progress conv=fsync
