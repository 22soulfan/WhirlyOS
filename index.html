The Stack

NT Compatibility

MyriadOS

Base: ReactOS Kernel

The primary open-source Windows alternative. MyriadOS (and its sister fork **openMyriad**) focuses on binary compatibility with Windows applications while providing a modern C++ shell and a signature Android-inspired lockscreen.

Rolling Release

RenardOS

Base: openSUSE Tumbleweed

Built for those who need the latest packages without sacrificing stability. RenardOS leverages the power of openSUSE Tumbleweed, optimized with a custom minimalist desktop environment for maximum speed.

Experimental C#

RPandaOS

Base: Cosmos C# DevKit

A groundbreaking experiment in OS design. RPandaOS is written using the Cosmos (C# Open Source Managed Operating System) toolkit, creating a managed code environment that is safe, fast, and visually vibrant.

Educational

WhirlyOS

Base: Debian Stable

The ultimate educational tool. Based on the rock-solid Debian kernel, WhirlyOS provides a distraction-free environment for students, featuring i386 support and cinematic, Pixar-inspired themes.

Anti-Microsoft

Communiverse11

Base: Windows 11 (Stripped)

A radical modification of Windows 11. Communiverse11 is "Anti-Microsoft"—meaning it strips away all telemetry, forced accounts, and bloatware, leaving only a high-performance core with minimal support.

Web Hybrid

ChromMyriad

Base: ChromiumOS

A cloud-centric OS that brings the Android security aesthetic to the web. Focused on providing a familiar mobile-style lockscreen for Chromium-based workflows.

WonderRepo Lab

Kinetic Shell

Custom UI logic for ReactOS (MyriadOS) using modern C++ for fluid animations.

Spark Engine

The C# implementation of our UI engine for RPandaOS using the Cosmos DevKit.

Zero-Trace

The stripping script used in Communiverse11 to remove Microsoft service dependencies.
