System Architectures

The Core

MyriadOS

The definitive standard of the project. MyriadOS serves as the technical blueprint for the "SoulFrame" desktop experience. It is designed to be a high-performance, modular base that influences the UI logic across the entire multiverse.

  • • Cinematic Shell architecture
  • • Unified Global Search API
  • • Cross-Distro configuration sync
Swift & Sleek

renardOS

Inspired by the cunning and swift nature of the fox. renardOS focuses on speed and hyper-minimalism. It strips away the heavy background processes of modern OSs to provide a lightning-fast environment for developers and power users.

  • • Aggressive RAM management
  • • FoxShell minimal interface
  • • Instant-Boot optimization
Bold Design

RPandaOS

A unique branch focusing on vibrant, expressive UI. RPandaOS (Red Panda) bridges the gap between fun aesthetic and functional utility. It is the testing ground for bold new color palettes and playful interactions within the Myriad Project.

  • • Vibrant Theme Engine
  • • Playful Widget architecture
  • • Touch-Friendly interaction logic
Modern NT

openMyriad

The open-source Windows 11 alternative. Built on a modernized ReactOS foundation, it brings an Android-inspired lockscreen and a hybrid Win 10/11 start menu to the FOSS world.

  • • Android-Style PIN/Pattern Lock
  • • Win 10/11 Hybrid Shell
  • • Binary Compatibility focus
Educational

WhirlyOS

The ADHD-aware, educational Linux distro. Based on Debian Stable, it uses cinematic Pixar-inspired themes to help students find their spark without the distraction of modern bloat or AI.

  • • i386 Legacy Hardware support
  • • Distraction-Free screen limits
  • • SoulFrame thematic UI
Cloud Sovereignty

ChromMyriad

Bringing the Android mobile experience to the desktop cloud. Featuring a native Android lockscreen and a stripped-down Chromium base for speed.

  • • Native Android Lock Integration
  • • Zero-Telemetry Chromium base
  • • Cloud-First modularity

The 22soulfan Manifesto.

The **Myriad Project** is a unified vision to decentralize how we interact with technology. By developing across different foundations (NT/ReactOS, Debian, Chromium), **22soulfan** ensures that no user is locked into a single ecosystem.

We believe in **Minimal Support** as a feature—reducing the overwhelming noise of constant updates and AI intrusions to focus on what matters: the user's passion.
