What is WhirlyOS?

A curated Debian-based OS that wraps a Linux base in a playful, ADHD-aware shell with browser-based learning tools.

Hardware Requirements?

Optimized for i386. Requirements vary from 512MB RAM (NotebookOS) to 4GB RAM (ElementalDesk).

Is it safe to dual-boot?

Warning: Install at your own risk. Breaking partitions can damage your device. Always do a clean install if unsure.
