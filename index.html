Myriad Multiverse Directory

MyriadOS

Base: ReactOS (NT-Binary)

The crown jewel. A custom-skinned ReactOS core designed to run .exe files without the bloat of modern Windows. Uses the "Spark" aesthetic engine.

WhirlyOS

Base: Debian Stable

Stripped to the bone. Every non-essential daemon is removed. Designed specifically for low-resource education laptops and $1\text{GB}$ machines.

RenardOS

Base: openSUSE Tumbleweed

Rolling release workstation. Integrates the Myriad Shell with cutting-edge Linux kernels for modern developers who want visual perfection.

RPandaOS

Base: Cosmos (C# Kernel)

Visual kernel experiment. Built in C# to explore managed-memory operating systems with high-level graphical abstractions.

ChromMyriad

Base: Gentoo / Chromium

The Cloud Warrior. Gentoo performance meets web-standard interfaces. Pure speed for users who exist entirely in the browser.

Comm11

Base: Windows 11 LTSC Mod

Surgically debloated. Removes Edge, Widgets, and Telemetry. Restores the "Light-Aesthetic" feel to the standard NT kernel.
