The Mission

To reclaim digital sovereignty by engineering high-performance operating environments that breathe life back into legacy hardware, proving aesthetics don't require high specs.

Strategic Goals

Eliminate telemetry at the kernel level, achieve a sub-200MB idle memory footprint, and maintain binary parity across NT and Linux distributions.

Core Benefits

Instant-boot capabilities, zero cloud dependency, unified visual language, and hardware reclamation of 2010-era devices.

About

Crafted by **22soulfan**, Myriad is an ongoing experiment in kernel optimization and C++ based shell architecture for the next decade.

The OS Collection

MyriadOS NT-Hybrid

Inspired by: Windows Longhorn (Early Concepts) & Windows 7 Aero

MyriadOS is a direct surgical modification of the ReactOS kernel, aimed at achieving 100% NT-binary compatibility without the telemetry burden of modern Windows. It utilizes a custom GDI rendering engine to simulate glass effects on hardware with no GPU acceleration. The kernel is stripped of all non-essential services, allowing it to idle at roughly 140MB of RAM. It features the "Spark" shell, which replaces the standard Explorer with a C++ based fluid interface. File management is handled through an integrated tabbed explorer with native .iso and .zip mounting. Driver support is focused on late-2000s hardware, reclaiming Dell Latitudes and ThinkPads. It is the flagship of the project, representing the perfect marriage of nostalgia and efficiency. Security is handled through a local-only firewall that blocks all outbound "pings" by default. It supports standard .exe and .msi installers through a custom compatibility layer. Designed for users who miss the "Windows" workflow but hate the Windows data-mining.

WhirlyOS Unix-Base

Inspired by: NeXTSTEP & Minimalist Alpine Linux

WhirlyOS is the educational backbone of the Myriad project, built upon a rock-solid Debian Stable core. It is optimized for users who need a distraction-free environment for coding and system administration. The boot sequence is parallelized to achieve desktop readiness in under 8 seconds on SSD hardware. It completely avoids systemd where possible, opting for a lightweight init system to save cycles. The default desktop environment is "Orbit," a tiled-floating hybrid built on the X11/Wayland bridge. It includes the Myriad Lab Notes app and the Whirly Shell as core system components. Software management is handled via the 'Whirl' package wrapper, which simplifies apt and flatpak commands. It is designed to run perfectly on machines with as little as 512MB of RAM. Privacy is guaranteed through a kernel-level sandbox for web browsers. WhirlyOS is for the purist who believes that a computer should be a tool, not a service.

The DE Library

Spark-Shell

Inspiration: MacOS Aqua & KDE Plasma 4

The flagship UI. Focuses on heavy translucency, rounded corners, and a centered "Dynamic Dock." Built in C++/Qt for MyriadOS and RenardOS.

Orbit-Desktop

Inspiration: NeXTSTEP & GNOME 2

The efficiency king. A minimalist top-bar with a vertical side-dock. Low CPU overhead, designed for the WhirlyOS stable release.

Panda-View

Inspiration: Windows Mobile & WebOS

A tile-based interface designed for touchscreens and small tablets. Integrated into the RPandaOS experimental branch.

Ghost-WM

Inspiration: i3wm & Suckless dwm

A keyboard-only tiling window manager for power users. No icons, no bloat, just pure screen real estate.
