Strategic Goals

1. $1\text{GB}$ Standard: Prove that high-end aesthetics can run on low-end legacy hardware through kernel stripping and C++ UI optimization.

2. The Unified UI: Develop a "Spark" shell that allows a user to switch from a Debian base to a ReactOS base without changing their workflow.

3. Data Autonomy: Build a "Zero-Call" ecosystem where the OS never communicates with external servers without explicit user consent.

The Multiverse Collection

01

MyriadOS

BASE: REACTOS (NT)

The flagship binary-compatible system. Runs Windows applications on a custom Myriad-Aesthetic kernel. Designed for absolute legacy speed.

Status: Alpha-0.4
02

WhirlyOS

BASE: DEBIAN STABLE

The education-first warrior. A "Zero-Aesthetic" build that prioritizes raw CPU cycles for compiling and learning. Perfect for $1\text{GB}$ laptops.

Status: Stable Release
03

RenardOS

BASE: OPENSUSE

A rolling-release powerhouse with the "Obsidian" shell theme. Best for developers who need the latest tools without the bloat of standard KDE/Gnome.

Status: Beta-Rolling
04

RPandaOS

BASE: COSMOS (C#)

An experimental C#-based kernel. Visualizing memory management in real-time. A sandbox for the next generation of managed operating systems.

Status: Experimental
05

ChromMyriad

BASE: GENTOO / WEB

A Gentoo-optimized web-first OS. Built for users who live in the browser but want the local privacy of the Myriad Shell.

Status: Custom Build
06

Comm11

BASE: WIN11 DE-BLOAT

A surgical modification of Windows 11. Removes 80% of background services and all telemetry for a pure gaming/creative experience.

Status: Production Mod
